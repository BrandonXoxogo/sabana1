ANTECEDENTES
El Lenguaje de Marcado de Hipertexto (HTML) fue creado en 1980 por el físico Tim Berners-Lee, uno de los inventores de la World Wide Web. Berners-Lee, que trabajaba en el CERN (Organización Europea para la Investigación Nuclear), propuso un nuevo sistema de hipertexto para compartir documentos. El objetivo era crear un entorno que permitiera a los científicos divulgar sus investigaciones para que sus colegas pudieran consultarlas con relativa facilidad. HTML se utiliza para estructurar y desplegar una página web y sus contenidos, como párrafos, listas, imágenes y tablas de datos. Desde su creación, HTML ha pasado de ser utilizado exclusivamente para crear documentos electrónicos a emplearse en diversas aplicaciones electrónicas especializadas como buscadores, tiendas "en línea" y banca electrónica. A lo largo de los años, se han desarrollado sucesivas versiones de HTML, como HTML 2, HTML 3.0, HTML 4.0 y HTML 5.0: HTML 2 A finales de 1993 se empezó a trabajar en la segunda versión de HTML, llamada HTML+, con la idea de diseñar un superconjunto de HTML que evolucionase poco a poco desde la primera versión. HTML 3.0 El borrador del estándar HTML 3.0 fue propuesto por el recién formado W3C en marzo de 1995. Las sucesivas actualizaciones (3.1 y 3.2) fueron de gran éxito en los primeros navegadores populares de Netscape y Mosaic. HTML 4.0 Aparece en 1997 y ya es una recomendación de la W3C. Implementa características como XForms 1.0 que no necesitan implementar motores de navegación incompatibles con algunas páginas web HTML. HTML 5.0 Es la versión más reciente, de 2006.
